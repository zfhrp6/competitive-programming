## about

[AHC014](https://atcoder.jp/contests/ahc014/tasks/ahc014_a)

## requirements

* python 3.8.2
* poetry

## usage

* install dependencies
  ```console
  poetry install
  ```

* test
  ```console
  make test
  ```

* run
  ```console
  make run
  ```

* update score
  ```console
  make scoring
  ```
